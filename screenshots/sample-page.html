A calm fixture page for checking grayscale rendering

This local sample uses varied typography, colorful blocks, and article-like layout without depending on a live news site.

Monoff should render this page in grayscale while preserving contrast, spacing, and the general reading experience. The content is intentionally neutral so screenshots remain stable between releases.

The page includes bright color, muted interface text, body copy, and a sidebar. That gives the extension a useful visual test without loading third-party scripts, advertisements, cookie prompts, or changing headlines.

A local fixture makes release screenshots repeatable and keeps the review package focused on the extension.

Because the screenshot page is served over localhost, the release script can apply the same built stylesheet used by the packaged extension.
